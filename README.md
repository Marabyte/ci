# Marabyte CI

![Docker Build Status](https://img.shields.io/docker/build/marabyte/ci.svg?style=flat-square)

This repo contains a Docker image with tools used for CI.

## Installed tools

- Node.js 16.3.0
- Yarn 1.13.0
- npm 7.19.0
- Python 3.10.0
- pip 21.1.2
- Brotli 1.0.5
- Chrome 91 Headless

## Installation

`docker pull marabyte/ci:1.2.2`
