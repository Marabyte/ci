# Marabyte CI

![Docker Build Status](https://img.shields.io/docker/build/marabyte/ci.svg?style=flat-square)

This repo contains a Docker image with tools used for CI.

## Installed tools

- Node.js 10.15.1
- Yarn 1.13.0
- npm 6.4.1
- Python 3.6.6
- pip 10.0.1
- Brotli 1.0.5
- Chrome 72 Headless
- Serverless 1.36.3

## Installation

`docker pull marabyte/ci`
